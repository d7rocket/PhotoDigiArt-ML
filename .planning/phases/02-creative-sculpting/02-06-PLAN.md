Steps for user to verify:
1. Launch the application: `python -m apollo7`
2. Load one or more photos via the library panel
3. Click "Extract Features" and wait for extraction to complete
4. Verify feature viewer shows color palette, edge map, and depth map for selected photo
5. Click "Simulate" button -- point cloud should come alive with particle motion
6. Adjust speed and turbulence sliders -- motion should respond in real-time
7. Try each force section: increase/decrease flow field, attraction/repulsion, SPH viscosity, gravity/wind
8. Check FPS counter in viewport corner -- should stay above 20 FPS
9. Toggle "Performance Mode" if FPS is low -- should improve smoothness
10. Press Space bar -- simulation should pause/resume
11. Enable bloom -- particles should have visible glow
12. Enable depth of field -- distant particles should blur
13. Enable motion trails -- particle paths should leave fading ghost trails
14. Adjust all post-fx sliders and verify visual changes
15. Press Ctrl+Z several times -- parameter changes should undo step by step
16. Press Ctrl+Shift+Z to redo
17. Click "Reset All" -- all parameters should return to defaults
18. Press Ctrl+S -- save project to a file
19. Close and reopen the application
20. Press Ctrl+O -- load the saved project, verify all parameters restored and point cloud visible
21. Save current params as a preset with a name and category
22. Reset all params, then apply the saved preset -- should restore
23. Press Ctrl+E -- export a 2x resolution PNG, verify the file is a valid image
24. Export with transparent background -- verify PNG has alpha channel

Aesthetic quality check: With bloom enabled, motion trails on, and flowing particles -- does the output look compelling and artistic? Not just functional, but genuinely visually interesting?
