<objective>
Wire the simulation engine (Plan 01) to the GUI with a full simulation controls panel, FPS counter, simulate button, keyboard shortcuts, and the render-then-interact flow. This plan connects the compute backend to the user experience.

Purpose: The simulation engine is useless without controls. This plan makes it interactive -- the user reviews their static point cloud, clicks "Simulate" to bring it alive, then tweaks parameters while particles flow. This is the core creative sculpting interaction loop.

Output: SimulationPanel with all force/speed/turbulence controls, FPS counter overlay, Simulate/Pause buttons, Space bar shortcut, performance mode toggle, per-section reset buttons, and all wiring between controls <-> engine <-> viewport.
</objective>

**SimulationPanel** (`simulation_panel.py`):
A panel with collapsible QGroupBox sections (per user decision, use collapsible sections for the thin right panel):

1. **Simulation Control section** (always visible at top):
   - "Simulate" QPushButton (large, prominent, accent colored) -- starts simulation
   - "Pause" / "Resume" toggle QPushButton -- only enabled when sim is running
   - "Performance Mode" QCheckBox -- reduces sim quality for smooth interaction
   - Speed slider (0.1 to 5.0, default 1.0)
   - Turbulence slider (0.0 to 3.0, default 1.0)
   - "Reset Simulation" section reset button

2. **Flow Field section** (collapsible QGroupBox):
   - Noise frequency slider (0.1 to 5.0, default 0.5)
   - Noise amplitude slider (0.0 to 3.0, default 1.0)
   - Noise octaves slider (1 to 8, default 4, integer)
   - "Reset Flow" button

3. **Forces section** (collapsible QGroupBox):
   - Attraction strength slider (0.0 to 2.0, default 0.5)
   - Repulsion strength slider (0.0 to 2.0, default 0.3)
   - Repulsion radius slider (0.01 to 0.5, default 0.1)
   - Gravity Y slider (-1.0 to 1.0, default -0.1)
   - Wind X/Z sliders (-1.0 to 1.0, default 0.0)
   - "Reset Forces" button

4. **Fluid (SPH) section** (collapsible QGroupBox):
   - Viscosity slider (0.0 to 1.0, default 0.1)
   - Pressure strength slider (0.0 to 5.0, default 1.0)
   - Surface tension slider (0.0 to 0.1, default 0.01)
   - "Reset Fluid" button
