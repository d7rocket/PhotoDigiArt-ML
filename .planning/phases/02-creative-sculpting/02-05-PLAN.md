<objective>
Add project save/load, high-resolution PNG export, and a categorized preset library. These are the "professional tool" features that let artists save their work, produce output, and build a library of reusable parameter configurations.

Purpose: Without save/load, work is lost when the app closes. Without export, there's no output artifact. Without presets, every session starts from scratch. These features transform Apollo 7 from a demo into a usable creative tool.

Output: Project files (JSON), PNG export at arbitrary resolution with alpha, preset library with categories and browse UI.
</objective>
