**Step 1: Create apollo7/gui/widgets/preset_card.py**

```python
class PresetCard(QtWidgets.QWidget):
```

Widget displaying a gradient thumbnail + preset name in a compact card:

- Fixed size: card fills grid cell width. Internal layout: QVBoxLayout with 0 margins, 0 spacing
- Gradient area: QLabel with QPixmap, fixed height 60px, full width
- Name label: centered text, 11px regular, color #808080, padding 4px 8px
- Card styling: bg #2d2d2d, border 1px solid #3a3a3a, border-radius 6px, overflow hidden
- Hover: border-color #808080
- Selected: border-color #0078FF, name color #e0e0e0

Properties:
- `preset_name: str` (read-only)
- `preset_data: dict` (stores the full preset dict)
- `selected: bool` (get/set, updates border style)

Signals:
- `clicked = Signal(str)` -- emits preset_name when card is clicked

Methods:
- `__init__(self, name: str, preset_data: dict, parent=None)`: Creates the card, generates gradient icon
- `_generate_gradient_icon(self, preset_data: dict) -> QPixmap`: Generates 80x60 gradient from preset params
  - Algorithm: Map key simulation params to gradient colors:
    - `solver_iterations` (1-6) maps to cohesion: low = spread gradient, high = concentrated
    - `noise_amplitude` (0-5) maps to intensity: low = muted blue tones, high = vibrant warm tones
    - `home_strength` maps to brightness: low = darker, high = lighter
  - Use QLinearGradient with 2-3 color stops derived from params
  - Base hue: electric blue (#0078FF) shifted by param values
  - Color stops: `QColor.fromHsvF(hue, saturation, value)` where:
    - hue = 0.58 + (noise_amplitude / 5.0) * 0.25 (blue -> purple range for artistic feel)
    - saturation = 0.4 + (solver_iterations / 6.0) * 0.4
    - value = 0.2 + (home_strength / 20.0) * 0.4
  - Second color stop shifts hue by +0.15
  - Paint onto QPixmap(width, height) using QPainter
- `mousePressEvent(self, event)`: Emit `clicked` signal with preset_name
- `set_selected(self, selected: bool)`: Update border to #0078FF if selected, #3a3a3a if not
