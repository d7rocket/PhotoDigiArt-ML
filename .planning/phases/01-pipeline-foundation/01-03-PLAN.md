  <action>
    **Base interface** — Create `apollo7/extraction/base.py`:
    - `ExtractionResult` dataclass: `extractor_name: str`, `data: dict[str, Any]` (scalars/metadata), `arrays: dict[str, np.ndarray]` (array data like edge maps)
    - `BaseExtractor(ABC)`: abstract property `name -> str`, abstract method `extract(image: np.ndarray) -> ExtractionResult`
    - Input images are always RGB float32 [0.0-1.0] numpy arrays (as returned by loader.load_image)

    **Color extractor** — Create `apollo7/extraction/color.py` with `ColorExtractor(BaseExtractor)`:
    - `name = "color"`
    - Uses `extcolors` for dominant color palette extraction. Convert float32 [0,1] back to uint8 [0,255] PIL Image for extcolors input.
    - Extract: dominant_colors (list of (R,G,B) tuples with occurrence counts), color histogram (per-channel via numpy), color distribution map (for visualization)
    - Returns ExtractionResult with `data={"dominant_colors": [...], "color_count": N}`, `arrays={"histogram": np.ndarray, "color_distribution": np.ndarray}`

    **Edge extractor** — Create `apollo7/extraction/edges.py` with `EdgeExtractor(BaseExtractor)`:
    - `name = "edge"`
    - Convert input to uint8 grayscale for OpenCV
    - Apply Canny edge detection (thresholds: low=50, high=150 — configurable)
    - Find contours via `cv2.findContours`
    - Returns ExtractionResult with `data={"contour_count": N}`, `arrays={"edge_map": np.ndarray (uint8 H,W), "contour_image": np.ndarray (uint8 H,W,3)}`

    **Feature cache** — Create `apollo7/extraction/cache.py` with `FeatureCache`:
    - In-memory dict keyed by `(photo_path, extractor_name)` tuple
    - Methods: `get(photo_path, extractor_name) -> ExtractionResult | None`, `store(photo_path, extractor_name, result)`, `invalidate(photo_path)` (clears all extractors for that photo), `clear()` (clears everything)
    - Per user decision: "Feature results cached after extraction — no re-processing unless user requests it"
