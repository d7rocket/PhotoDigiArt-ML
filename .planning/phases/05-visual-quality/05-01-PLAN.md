<objective>
Eliminate the CPU readback bottleneck by sharing GPU buffers directly between PBF compute and pygfx rendering.

Purpose: The current `_update_points_from_sim()` calls `read_positions()` which does `device.queue.read_buffer()` -- this stalls the GPU pipeline and caps particle throughput. GPU buffer sharing enables 1M+ particles at 60fps.

Output: Extract-positions compute shader, shared render buffer with VERTEX usage, pygfx buffer injection in viewport, and test scaffolds for all Phase 5 visual quality requirements.
</objective>
