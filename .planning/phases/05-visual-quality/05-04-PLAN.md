2. Create `apollo7/rendering/crossfade.py` with:

```python
class CrossfadeEngine:
    TICK_MS = 16          # ~60fps update rate
    DURATION_MS = 400     # ~0.4s transition (user wants 0.3-0.5s)
    DISCRETE_PARAMS = frozenset({"solver_iterations"})
```

Implementation:
- `__init__(self, apply_fn: Callable[[str, float], None])` -- callback receives (param_name, interpolated_value)
- Internal `_active: dict[str, _Transition]` tracking active transitions
- `_Transition` dataclass: `start: float, end: float, progress: float` (0.0 to 1.0)
- `set_target(name: str, target: float, current: float)` -- if name is in DISCRETE_PARAMS, call apply_fn immediately and return. Otherwise create/update transition.
- If transition for `name` already active (retarget mid-transition), compute current interpolated value as new start, reset progress to 0.0, set new end.
- `_tick()` method: iterate active transitions, advance progress by TICK_MS/DURATION_MS, compute eased value using cubic ease-out `1 - (1-t)^3`, call apply_fn, remove completed transitions, stop timer when empty.
- QTimer starts on first set_target, stops when _active is empty.

Per user decision: ease-out curve, fast initial response, smooth deceleration. Unified system for both sliders and preset CrossfadeWidget.
