    **Major rewrite of engine.py** to use PBFSolver instead of the old 3-pipeline SPH approach.

    Changes to SimulationEngine.__init__:
    - Remove: _integrate_pipeline, _integrate_bind_group, _integrate_bgl
    - Remove: _forces_pipeline, _forces_bgl, _forces_bind_group
    - Remove: _sph_density_pipeline, _sph_force_pipeline, _sph_bgl, _sph_bind_group
    - Remove: _performance_mode, _sim_steps_per_frame (PBF is always full pipeline)
    - Add: _pbf_solver: PBFSolver | None = None
    - Keep: _device, _state, _params, _particle_buffer, _initial_positions, _initial_colors, _time
    - Keep: _attractors, _attractor_buffer, _max_attractors (future use)
    - Keep: _has_feature_textures

    Changes to initialize():
    - After ParticleBuffer upload, apply feature texture modulation to home positions:
      If feature_textures provided with "edge_map", sample edge values for each particle's UV coordinate and write feature_strength into home_positions.w. Particles on edges get w=1.5 (hold tighter), flat areas get w=0.5 (drift more). Use the existing _upload_feature_textures for GPU textures, but also compute the per-particle feature_strength on CPU during init and upload to home_positions_buffer.w.
    - Create PBFSolver: self._pbf_solver = PBFSolver(self._device, self._particle_buffer)
    - Remove calls to _build_forces_pipeline, _build_sph_pipelines, _build_integrate_pipeline

    Changes to _step_once():
    - Remove all old dispatch code (forces, SPH density, SPH force, integrate)
    - Replace with:
      1. Update time: self._time += self._params.dt
      2. Compute breathing modulation: breathing_mod = self._params.compute_breathing(self._time)
      3. Compute CFL-adaptive dt (see below)
      4. Update params with time, breathing_mod, and adaptive dt
      5. Call self._pbf_solver.step(updated_params)
    - Remove: _dispatch_chunked (moved to PBFSolver), _rebuild_*_bind_group methods

    CFL-adaptive timestep:
    - After each PBF step, read max velocity from a small buffer (the PBFSolver should expose a method or the engine can compute it).
    - For initial implementation: use a conservative fixed dt (0.0083 from research) without GPU readback. The CFL readback can be added as an enhancement -- reading a single u32 is acceptable.
    - Alternative simpler approach: PBFSolver.step() can take a max_velocity_callback that the engine provides. Or, since the finalize shader can write max velocity to an atomic buffer, add a max_velocity_buffer to ParticleBuffer and read it back (4 bytes).
    - Implement: Add max_velocity_buffer (single u32) to ParticleBuffer. In pbf_finalize.wgsl, atomicMax the speed as bitcast<u32>. After PBF step, read back 4 bytes, bitcast to float, compute dt = min(dt_target, CFL_COEFF * h / max(v_max, 0.001)). CFL_COEFF = 0.4.
    - If bitcast+atomicMax causes issues on AMD (open question from research), fall back to fixed dt.

    Changes to step():
    - Remove _sim_steps_per_frame logic. Just call _step_once().

    Changes to restart():
    - After re-uploading positions, rebuild PBFSolver (or if PBFSolver is stateless w.r.t. particles, just let it work).
    - Ensure home_positions are re-uploaded from initial positions.

    Keep unchanged: get_positions_buffer(), get_colors_buffer(), pause(), resume(), toggle_pause(), update_visual_param(), update_physics_param(), particle_count, params, state properties.

    Remove: set_performance_mode(), set_sim_steps_per_frame() -- PBF does not need these.

    Remove methods: _build_forces_pipeline, _build_sph_pipelines, _build_integrate_pipeline, _rebuild_forces_bind_group, _rebuild_sph_bind_group, _rebuild_integrate_bind_group, _dispatch_chunked, _extract_noise_functions.

    **Delete old shader files:**
    - Delete apollo7/simulation/shaders/integrate.wgsl
    - Delete apollo7/simulation/shaders/forces.wgsl
    - Delete apollo7/simulation/shaders/sph.wgsl

    Keep: noise.wgsl (shared noise functions used by pbf_predict), flow_field.wgsl (will be used in Plan 04).
