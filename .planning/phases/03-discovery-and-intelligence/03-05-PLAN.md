- DimensionalMapper:
  - _values: dict[str, float] = {"energy": 0.5, "density": 0.5, "flow": 0.5, "structure": 0.5}
  - set_dimension(name: str, value: float) with exponential smoothing (alpha=0.3)
  - get_constraints() -> dict[str, tuple[float, float]]:
    - For each dimension, compute constrained ranges based on slider position
    - At value=0: use the lower portion of the range
    - At value=1: use the upper portion
    - Overlap between dimensions is resolved by intersection (narrower range)
  - get_param_ranges() -> dict mapping param_name -> (min, max) for RandomWalk

Create apollo7/discovery/random_walk.py:
- RandomWalk:
  - __init__(seed: int | None = None) for reproducibility option
  - _rng: numpy random Generator
  - propose(constraints: dict[str, tuple[float, float]], current: SimulationParams | None = None) -> SimulationParams:
    - If current is None: pure random within constraints
    - If current provided: perturb current values by +/- 10-30% within constraints (random walk step)
    - For each constrained param: uniform random in [min, max]
    - For unconstrained params: use sensible defaults
    - For tuple params (gravity, wind): randomize each component independently within [-1, 1]
    - Return new SimulationParams via with_update()
  - _perturb(current_value, min_val, max_val, step_size=0.2) -> float:
    - Add gaussian noise scaled by step_size * (max - min), clamp to range
