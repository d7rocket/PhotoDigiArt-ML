  <action>
Create apollo7/api/__init__.py with module docstring.

Create apollo7/api/enrichment.py:
- EnrichmentResult dataclass:
  - description: str | None (artistic mood description)
  - suggestion: str | None (sculpting suggestion)
  - mapping_suggestions: list[dict] | None (suggested feature-to-param connections)

- EnrichmentService:
  - __init__(api_key: str | None = None)
  - _client: anthropic.Anthropic | None (lazy-created)
  - enrich_tags(image_path: str, basic_tags: list[tuple[str, float]]) -> EnrichmentResult | None:
    - If no api_key: return None immediately
    - Load image, base64 encode
    - Call client.messages.create() with claude-sonnet-4-20250514:
      - System: "You are an art advisor analyzing photos for a 3D data sculpture tool."
      - User message: image + "These tags were detected: {tags}. Provide JSON: {description, suggestion}"
      - max_tokens: 256
    - Parse JSON response
    - Return EnrichmentResult
    - On ANY exception: log warning, return None (offline-first)
  - suggest_mappings(tags: list[tuple[str, float]], available_params: list[str]) -> list[dict]:
    - If no api_key: return [] immediately
    - Call Claude with text-only prompt (no image needed):
      - "Given these semantic tags {tags} and these visual parameters {params}, suggest 3-5 creative mappings. Respond as JSON array of {source_key, target_param, strength, reasoning}."
    - Parse response, return list of dicts
    - On ANY exception: return []

- EnrichmentWorker(QRunnable):
  - Wraps EnrichmentService calls in background thread
  - Signals: enrichment_ready(EnrichmentResult), suggestions_ready(list)
  - Uses WorkerSignals QObject pattern from existing workers

- Load API key from environment: os.environ.get("APOLLO7_CLAUDE_API_KEY", None)
