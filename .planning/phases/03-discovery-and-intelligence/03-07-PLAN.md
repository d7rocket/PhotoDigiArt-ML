   - Add DiscoveryPanel to the right sidebar (below or as tab alongside SimulationPanel)
   - Wire signals:
     - discovery_panel.proposal_requested -> create RandomWalk, call propose() with DimensionalMapper constraints, apply result
     - discovery_panel.proposal_applied(dict) -> apply params to SimulationEngine and PostFX
     - discovery_panel.dimension_changed(str, float) -> update DimensionalMapper
   - On proposal applied: capture viewport thumbnail for history strip
   - Keyboard shortcut: Ctrl+D to toggle discovery panel focus

2. **Mapping Editor Overlay** (from Plan 03):
   - Add PatchBayEditor as a full overlay widget (hidden by default)
   - Toggle button in toolbar or menu: "Feature Mapping" (Ctrl+M shortcut)
   - Wire signals:
     - patch_bay.mapping_changed(MappingGraph) -> store graph, evaluate against current features, apply to params
   - On extraction complete: re-evaluate mapping graph with new feature data
   - On mapping change: apply MappingEngine.evaluate() results to SimulationParams

3. **Crossfade Widget** (from Plan 02):
   - Wire preset_panel crossfade_changed signal to apply lerped params:
     - crossfade_changed(dict) -> update SimulationParams and PostFX from lerped dict

4. **Animation System** (from Plan 02):
   - Create ParameterAnimator instance in MainWindow
   - Add animation tick to the viewport render loop (call animator.tick(time, params) each frame if active)
   - Wire animation bindings from mapping editor (optional -- animation bindings can be created through the mapping editor or a dedicated animation panel later)

5. **Collection Analysis Trigger** (from Plan 04):
   - After ALL photos in a batch have been extracted (extraction_complete signal):
     - Gather all CLIP embeddings from cached ExtractionResults
     - Run CollectionAnalyzer.analyze() in a QRunnable worker
     - On result: update EmbeddingCloudManager in viewport, set_attractors() in SimulationEngine
   - Add toggle for embedding cloud visibility (Ctrl+E or toolbar button)
   - Wire click-to-isolate from EmbeddingCloudManager

6. **Enrichment Service** (from Plan 06):
   - Create EnrichmentService instance with API key from settings
   - Wire feature_viewer.enrichment_requested -> EnrichmentWorker -> feature_viewer.set_enrichment
   - Respect ENRICHMENT_ENABLED setting

7. **Project State Persistence**:
   - Extend ProjectState (apollo7/project/state.py) to include:
     - mapping_graph: MappingGraph serialized dict
     - discovery_dimensions: dict[str, float] (current slider positions)
     - enrichment_enabled: bool
   - Update save/load to persist and restore these new fields
   - Ensure backward compatibility: loading old project files without these fields works (use defaults)

8. **Menu/Toolbar Updates**:
   - Add "Intelligence" menu or section:
     - Toggle Discovery Mode (Ctrl+D)
     - Open Feature Mapping (Ctrl+M)
     - Toggle Embedding Cloud (Ctrl+E)
