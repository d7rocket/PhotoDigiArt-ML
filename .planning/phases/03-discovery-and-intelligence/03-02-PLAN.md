    - Envelope.evaluate(time) produces attack-sustain-release shape with configurable durations
    - AnimationBinding(target_param, source, min_val, max_val) maps generator output to a parameter range

Create apollo7/animation/__init__.py with module docstring.

Create apollo7/animation/lfo.py:
- LFO class with frequency, amplitude, offset, waveform (sine/triangle/square/sawtooth)
  - evaluate(time: float) -> float
  - Sine: math.sin(2*pi*freq*time) * amp + offset
  - Triangle: 2*abs(2*(time*freq % 1) - 1) - 1 scaled
  - Square: sign of sine
  - Sawtooth: 2*(time*freq % 1) - 1 scaled
- NoiseGenerator class with frequency, amplitude, seed
  - evaluate(time: float) -> float using hash-based deterministic noise with linear interpolation between integer time steps for smoothness
- Envelope class with attack, sustain, release durations and peak amplitude
  - evaluate(time: float) -> float
  - Returns 0 before trigger, ramps to peak during attack, holds during sustain, decays during release

Create apollo7/animation/animator.py:
- AnimationBinding dataclass: target_param (str), source (LFO|NoiseGenerator|Envelope), min_val (float), max_val (float)
  - The source output is mapped from [-1, 1] range to [min_val, max_val]
- ParameterAnimator class:
  - _bindings: list[AnimationBinding]
  - add_binding(binding) / remove_binding(target_param)
  - tick(time: float, params: SimulationParams) -> SimulationParams
    - For each binding: evaluate source at time, map to [min_val, max_val], collect updates
    - Return params.with_update(**updates)
  - is_active property: True if any bindings exist

- test_noise_deterministic: same seed+time produces same output

- test_envelope_shape: attack ramps up, sustain holds, release decays
- test_animator_applies_binding: tick returns params with updated value
- test_animator_multiple_bindings: multiple params animated simultaneously
- test_animator_empty: tick with no bindings returns unchanged params
